# 视觉设计规格 — KOF2000 风格

**版本：** v0.2.0
**更新日期：** 2026-02-22

---

## 1. 设计目标

以 King of Fighters 2000 为参考，对游戏的 UI、角色渲染、舞台背景进行全面升级，
替换所有临时占位素材（彩色方块、纯色背景、测试音频）。

## 3. HUD 布局

```
┌─────────────────────────────────────────────────────────────┐
│ [P1]  ████████████████████░░░  [●●○]  99  [○○●]  ███████ [P2]│
│       ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │
│       [■■■□]                              [□■■■]            │
└─────────────────────────────────────────────────────────────┘
```

### 3.1 血条

- **位置**：屏幕顶部，距顶 8px
- **P1**：左侧，x=20px，宽度 = 屏幕宽 × 0.38，从左向右填充
- **P2**：右侧，x = 屏幕宽 - 20 - 宽度，从右向左填充（镜像）
- **高度**：18px
- **颜色渐变**：
  - HP > 50%：绿色 `#10c030`
  - HP 25–50%：黄色 `#d0b000`
  - HP < 25%：红色 `#d01010`（同时开始闪烁，6帧周期）
- **边框**：金色 `#c8a000`，1px，四边
- **背景**：深色 `#111118`
- **白色延迟条**：在血条后方，30帧延迟后以 0.5%/帧 速度消耗

### 3.2 角色名称

- **位置**：血条正上方，距血条 2px
- **字体大小**：12px
- **颜色**：白色 `#ffffff`，有 1px 黑色描边
- **P1**：左对齐；**P2**：右对齐

### 3.3 回合胜利标志

- **位置**：血条正下方，距血条 3px
- **形状**：圆形，直径 8px
- **已赢**：实心金色 `#c8a000`
- **未赢**：空心，边框金色，内部透明
- **间距**：圆点间距 4px

### 3.4 能量槽（Power Gauge）

- **位置**：回合标志下方，距 3px
- **格数**：3 格
- **单格尺寸**：宽 = 血条宽 / 3 - 4px，高 = 8px
- **间距**：4px
- **颜色**：
  - 空格：深色 `#0a0a20`
  - 部分填充：蓝色 `#1a6fff`
  - 满格：亮蓝 `#40aaff`，有发光效果（外扩 2px 半透明蓝色）

### 3.5 计时器

- **位置**：屏幕顶部居中
- **字体大小**：36px，粗体
- **背景框**：深色 `#111118`，圆角 4px，内边距 8px × 4px
- **边框**：金色 `#c8a000`，1px
- **颜色**：金色 `#e8c000`
- **紧急状态**（≤ 10 秒）：红色 `#e82020`，每秒闪烁一次

## 5. 回合流程动画

### 5.1 回合开始序列

```
帧 0-30:   "ROUND X" 文字从中心放大（scale 0.5 → 1.0）
帧 30-60:  "ROUND X" 保持显示
帧 60-90:  "FIGHT!" 文字出现，"ROUND X" 淡出
帧 90-120: "FIGHT!" 保持显示
帧 120+:   所有文字淡出，战斗开始
```

### 5.2 回合结束序列

```
帧 0-30:   "K.O." / "TIME OVER" / "DRAW" 文字出现
帧 30-120: 保持显示（2秒冻结）
帧 120+:   进入下一回合或比赛结束
```

## 6. 角色精灵规格

### 6.1 精灵表格式

- **格式**：PNG，RGBA
- **布局**：横向排列，每行一个动作
- **帧大小**：120 × 120px（标准 KOF 尺寸）
- **命名**：`{character}_{action}_{frame}.png` 或精灵表 `{character}.png`

### 6.2 必需动画

| 动作 | 帧数 | 帧时长（逻辑帧） |
|------|------|-----------------|
| Idle | 4 | 8 |
| WalkForward | 8 | 4 |
| WalkBackward | 8 | 4 |
| Jump | 6 | 4 |
| Crouch | 2 | 4 |
| Attack_A | 6 | 3 |
| Attack_B | 8 | 3 |
| Hitstun | 3 | 4 |
| Knockdown | 6 | 4 |
| KO | 8 | 5 |

### 6.3 精灵对齐

- **原点**：角色脚底中心
- **朝向**：默认面向右（P2 渲染时水平翻转）

## 7. 舞台背景规格

### 7.1 层次结构

```
Layer 0 (最远): 天空/远景，视差系数 0.1
Layer 1:        中景建筑，视差系数 0.3
Layer 2:        近景元素，视差系数 0.6
Layer 3 (地面): 地板纹理，视差系数 1.0
```

### 7.2 图片规格

- **格式**：PNG，RGBA（支持透明）
- **宽度**：≥ 1600px（支持横向滚动）
- **高度**：各层按需，总高约 400px

---

## 8. 音频规格

### 8.1 背景音乐

- **格式**：OGG Vorbis
- **比特率**：128–192 kbps
- **循环**：无缝循环（loop point 标记）
- **时长**：≥ 60 秒

### 8.2 音效

| 音效 | 文件名 | 时长 |
|------|--------|------|
| 轻攻击命中 | `hit_light.ogg` | ~0.1s |
| 中攻击命中 | `hit_medium.ogg` | ~0.15s |
| 重攻击命中 | `hit_heavy.ogg` | ~0.2s |
| 格挡 | `block.ogg` | ~0.1s |
| KO | `ko.ogg` | ~1.0s |
| 回合开始 | `round_start.ogg` | ~0.5s |
| 菜单选择 | `menu_select.ogg` | ~0.1s |
| 菜单确认 | `menu_confirm.ogg` | ~0.2s |
