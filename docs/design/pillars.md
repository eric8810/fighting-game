# Tickle — 设计宪法

> 所有设计决策的最终裁决依据。纠结时回到这里。

---

## 体验支柱

1. **连锁打击的演出感** — 连续命中时视觉、音效、受击反馈连成一片的爽感
2. **沉重的打击感** — 每一下都有分量，对手被打中的反应重而实
3. **丰富的机动选项** — 多种方式控制距离和接近对手，不是单纯的快，而是选项多

---

## 五维定位

### 空间博弈：远近切换

拉远牵制和突然贴身交替出现。对战不会一直在某个距离上停留，距离变化本身就是战术的一部分。

### 风险/回报：大开大合

读对一次赚大的，读错也痛。一次成功的读取能造成显著伤害，一次失败的冒险也要承受严重后果。比赛节奏快，不是靠慢慢磨血取胜。

### 势头与翻盘：随时可翻

领先不安全，落后不绝望。落后方拥有有意义的逆转工具。配合大开大合的风险/回报，一次读取就可能逆转局面。

### 表达空间：双高

连段路线有创造力——同样的起手，不同玩家打出不同路线和伤害。战术选择有深度——资源管理、时机判断、距离控制因人而异。操作上限高，面向愿意投入时间钻研的玩家。

### 张力曲线：玩家主动点燃

高潮时刻不是系统自动给的，是玩家选择在某个时刻"赌一把"主动制造的——发动爆气、压上资源、选择高风险行动。这让每场比赛的戏剧高潮出现在不同的时间点，由玩家的胆量和判断决定。

## 竞品定位：以 KOF XIII 为基准

核心手感对标 KOF XIII，一处关键差异：

| 维度 | KOF XIII | Tickle |
|------|----------|--------|
| 空间博弈 | 远近切换都好 | 保持 |
| 风险/回报 | HD连段60-70%血，大开大合 | 保持 |
| 势头与翻盘 | HD+NEO MAX逆转 | 保持 |
| 表达空间 | HD连段自由度高 | **更深——角色差异更极端，每个角色玩法逻辑不同** |
| 张力曲线 | HD发动时机=玩家主动点燃 | 保持 |

一句话：**KOF XIII 的手感 + Guilty Gear 级别的角色差异化。**

## 核心机制决策

基于五维定位，Tickle 需要以下机制来实现目标体验。

### 空间博弈 → 通用移动 + 角色专属移动

- **通用移动**：KOF XIII 框架（步行、奔跑、后退、小跳/普通跳/大跳、前后滚）
- **角色专属移动**：通过必杀技/特殊技实现位移特性，不是额外加按键
- **设计原则**：必杀技不只是伤害工具，也是移动工具。每个角色至少有一个"定义他怎么控制距离"的招式
- 参考：K' 的 Minute Spike 滑行突进、八神的葵花位移压制

### 风险/回报 → 全面大开大合

- **连段高伤**：保持 KOF XIII 的程度（一套连段 60-70% 血）
- **单发也重**：重攻击、超必杀裸中也造成可观伤害，不接连段也痛
- **防御选择有风险**：主动防御选项（类似 Blocking）赌对了大赚，赌错了比老实防御还惨
- 不只是连段系统大开大合，整个攻防系统都是高风险高回报的

### 势头与翻盘 → 攻防双通道逆转

- **进攻端逆转**：攒资源 → 爆发模式 → 高伤害连段翻盘
- **防御端逆转**：被压制时选择高风险防御选项，成功则反击
- **两条路都有**：落后方选择赌进攻还是赌防御，领先方也要猜对手走哪条路

### 表达空间 → 角色设计阶段实现

- 连段自由度：至少达到 KOF XIII HD 连段的水平
- 角色差异化：Guilty Gear 级别——每个角色的玩法逻辑本身不同，不只是换招式
- 在角色设计阶段（阶段三）落实

### 张力曲线 → 单一资源的攻防博弈

- **单一资源系统**：进攻爆发（HD 类模式）和防御赌博（高风险防御选项）消耗同一条气
- 核心纠结："我是把气存着防守翻盘，还是现在就赌一波进攻？"
- 这个资源选择本身就是"玩家主动点燃"的载体——什么时候用、用在攻还是防，由玩家决定
